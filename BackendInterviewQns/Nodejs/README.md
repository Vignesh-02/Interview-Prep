real-world scenarios, and cross-language tradeoffs (Go, Python, Java, .NET).

After completing all topics, you will have the depth and breadth of a **senior backend engineer**.

## Question Format

Each file follows this structure:
- **Topic introduction** with illustration
- **Q1–Q5**: Beginner — fundamentals with code examples
- **Q6–Q12**: Intermediate — production patterns with scenarios
- **Q13–Q20**: Advanced — senior-level architecture, debugging, tradeoffs

Every question includes a **scenario or code example** (except purely theoretical ones).
Cross-language tradeoffs (Go/Python/Java/.NET) appear where they add value.
