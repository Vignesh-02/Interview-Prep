# React.js Interview Questions — Senior Level Prep

Master React 18/19 from beginner to senior level. 30 topics × 20 questions = **600 questions** covering everything from JSX basics to React Server Components and the React 19 Compiler.

## Format

Each topic file contains:
- **Q1-Q5**: Beginner — fundamentals every React dev should know
- **Q6-Q12**: Intermediate — production patterns, performance, real-world scenarios
- **Q13-Q20**: Advanced — senior-level architecture, deep internals, system design

Every question includes code examples, production scenarios, and focuses on React 18 (with React 19 where noted). After completing all topics, you should be able to answer any React interview question at a senior level.

## Key Concepts Covered

- **React 18**: Concurrent rendering, Suspense, useTransition, useDeferredValue, Streaming SSR
- **React 19**: React Compiler, Actions, useActionState, useFormStatus, useOptimistic, use() API, native metadata
- **Performance**: Virtualization, code splitting, lazy loading, memoization, Profiler
- **Architecture**: Server Components, SSR, hydration, micro-frontends, module federation
- **State Management**: useState, useReducer, Context, Redux Toolkit, Zustand, TanStack Query
- **Testing**: React Testing Library, user-event, MSW, component/integration/E2E testing
- **TypeScript**: Generics with React, discriminated unions, type-safe hooks and context
- **Production**: Error boundaries, monitoring, accessibility, security, deployment strategies
