# SQL Interview Questions — Top 20 Core + 10 Advanced Topics

A structured set of **20 core topics** and **10 advanced/senior-level topics**, with **10 questions each** (300+ questions total). Problem-solving focused; each topic includes **1–2 production-scenario** questions. Mix: **5 beginner/intermediate** (Q1–5) and **5 advanced** (Q6–10). Covers standard SQL with notes for **PostgreSQL**, **MySQL**, and **Oracle** where they differ, plus backend integration and DB comparison.
