# DevOps Interview Questions — Top 15 Core + 15 Advanced Topics

A structured set of **15 core topics** and **15 advanced/senior-level topics**, with **20 questions each** (600+ questions total). Problem-solving focused; each topic includes **4–5 production-scenario** questions with correct approach and answer. Mix: **10 beginner/intermediate** (Q1–10) and **10 advanced** (Q11–20). Includes **tradeoffs** (startup vs medium vs enterprise), **scenarios with user counts**, **actual code/commands** (shell, Docker, Kubernetes, Terraform, Jenkins, etc.), and **senior red flags to avoid** in production. Covers **Linux**, **shell scripting**, **system administration**, **Git**, **Docker**, **Kubernetes**, **CI/CD**, **monitoring**, **SLI/SLO/SLA**, **IaC**, **security**, and **reliability**.
